3. Privacy Policy — No Data Collection

PGPhone is designed with a privacy-first, offline-first architecture:

  • All cryptographic operations run entirely on your device, in your browser, with no server involvement.
  • Your keys, messages, and passphrases never leave your device. They are stored only in your browser's local IndexedDB.
  • The application makes zero network requests after installation. The Service Worker blocks all external connections.
  • The author does not collect, process, or store any personal data, usage data, analytics, or telemetry of any kind.

If you install PGPhone from a public website: the web server that hosts the files may log your IP address and standard HTTP connection metadata (timestamp, browser user-agent, bytes transferred) as part of normal server operation.
